# M.CSS

Un framework CSS minimalista diseñado para cubrir el 80% de los casos de uso en desarrollo web moderno, con solo **~10KB** minificado y sin dependencias externas.

## Qué incluye

- Reset CSS moderno
- Variables CSS nativas (colores, espaciado, tipografía, bordes)
- Estilos base semánticos para elementos HTML
- Clases utilitarias: flexbox, grid, spacing, tipografía, colores, bordes, display
- Diseño responsive mobile-first con breakpoints `sm` (640px) y `md` (768px)
- Accesibilidad integrada (focus visible, `.sr-only`, formularios semánticos)

## Uso

```html
<link rel="stylesheet" href="m.min.css">
```

Descarga `m.min.css`, inclúyelo en tu proyecto y empieza a usar las clases utilitarias.
